# Política de Privacidad (v1.0)

Este documento describe la política de privacidad de la aplicación.

## 1. Datos que recolectamos
- Datos personales que el usuario proporciona: nombre, correo electrónico, teléfono, dirección.
- Datos técnicos y de navegación: cookies, IP, agente de usuario.

## 2. Uso de los datos
- Procesamiento de pedidos.
- Comunicación con el cliente.
- Cumplimiento de obligaciones legales.

## 3. Conservación
- Se conservarán mientras la cuenta exista y los requerimientos legales lo exijan.

## 4. Consentimiento de cookies
- El consentimiento se registra en la BD para usuarios autenticados o en una cookie para invitados.
- La cookie de consentimiento tiene duración de 1 año.
